## 🌐 Visão Geral
Sistema de gerenciamento de bootcamps educacionais que permite:
- Cadastro de cursos e mentorias
- Inscrição de desenvolvedores
- Acompanhamento de progresso
- Cálculo automático de experiência (XP)
